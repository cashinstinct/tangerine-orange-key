Questions fréquentes

Foire aux questions

La Clé Orange 14130944S1 est-elle cumulable avec la prime de 250 $ ?

Oui. Les deux primes sont indépendantes et cumulables. La Clé Orange est une prime d'acquisition activée par le code 14130944S1 à l'inscription. La prime de 250 $ est une prime de comportement bancaire activée par le dépôt de paie — aucun code n'est requis pour cette dernière.

Où exactement faut-il entrer la Clé Orange ?

Dans le champ « Entrez une Clé Orange » à la première étape du formulaire d'inscription Tangerine — avant de choisir votre type de compte, avant les informations personnelles, avant le paiement. C'est la toute première écran du processus d'inscription.

Combien de temps pour recevoir la prime de 50 $ ?

Environ 3 mois au total : 30 jours pour effectuer le dépôt initial, 60 jours de maintien du solde à 250 $, puis 30 jours pour que Tangerine crédite la prime. Le versement est automatique — aucune démarche requise.

Comment activer la prime de dépôt de paie de 250 $ ?

Aucun code requis. Après avoir ouvert votre compte avec la Clé Orange, redirigez votre dépôt de paie direct vers Tangerine (institution 614, transit 00152, minimum 200 $/mois). Le premier dépôt doit arriver dans les 60 jours suivant l'ouverture. Après deux dépôts consécutifs, Tangerine verse les 250 $ automatiquement le mois suivant.

Tangerine est-elle disponible au Québec ?

Oui. Tangerine offre l'ensemble de ses services bancaires au Québec. L'application mobile, le site web et le service client sont entièrement disponibles en français.

Quels sont les codes bancaires Tangerine pour le dépôt de paie ?

Pour configurer un dépôt direct, fournissez à votre employeur : Numéro d'institution : 614 et Numéro de transit : 00152. Votre numéro de compte personnel est accessible dans l'application mobile Tangerine, sous les détails de votre compte.

Qu'est-ce qu'une Clé Orange Tangerine (ou Cle Orange) ?

La « Clé Orange » (parfois orthographiée « Cle Orange » sans accent) est le terme officiel de Tangerine pour son code de parrainage. Entrer 14130944S1 dans le champ prévu active la prime de bienvenue de 50 $. C'est l'équivalent d'un code promo chez d'autres banques, mais spécifique au système de parrainage Tangerine.
